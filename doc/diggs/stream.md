## Stream

Displays information from stream ( "mikroblog" )

####  stream_show

Displays stream

```
client.stream_show

# Displays page 3 of the stream
client.stream_show(3)
```

####  stream_hot

Displays stream from 'hot' category

```
client.stream_hot

# Displays page 3 of the stream
client.stream_hot(3)
```
